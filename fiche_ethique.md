# Fiche éthique — Détection de discours haineux

## Principes

1. **Humain dans la boucle** : l'IA signale, mais une personne décide pour les cas importants.
2. **Vie privée** : minimiser les données, anonymiser les journaux, éviter les données sensibles inutiles.
3. **Transparence** : expliquer pourquoi un message est signalé.
4. **Droit à l'erreur et à l'appel** : permettre la contestation d'une décision.
5. **Anti-biais** : mesurer les erreurs selon les groupes, dialectes, langues et contextes.

## Usage recommandé

- Prioriser les messages à réviser.
- Protéger les communautés contre le harcèlement.
- Aider les modérateurs à gérer un grand volume de messages.

## Usage déconseillé

- Sanction automatique sans révision.
- Surveillance de personnes sans cadre légal clair.
- Classement de personnes plutôt que de contenus.
